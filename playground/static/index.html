Gleam Playground

Hello, friend!

Welcome to the Gleam Playground!

The Gleam code on the left is editable and will be compiled and evaluated as you type. Anything you print using io.println or io.debug will be shown in the lower right, along with any compile errors and warnings.

To compile your code this playground has a WASM version of the Gleam compiler which is used to compile Gleam to JavaScript code, which is then evaluated in the browser.

The Gleam standard library is loaded into the playground, so you can import any of the modules it contains. gleam/list and gleam/result are Gleamlins favourites.

Happy hacking!
