# project

A Gleam project

```sh
gleam run   # Run the project
gleam test  # Run the tests
gleam shell # Run an Erlang shell
```

## Installation

If available on Hex this package can be installed by adding `project` 
to your `gleam.toml` dependencies:

```erlang
[dependencies]
project = "~> 0.1.0"
```
